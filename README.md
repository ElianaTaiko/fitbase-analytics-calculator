# Калькулятор тарифов Fitbase Analytics

Внутренний инструмент отдела продаж для расчёта стоимости подключения дашбордов Fitbase Analytics.

## Запуск

Вариант 1 — просто открыть `index.html` двойным кликом (работает офлайн, без сервера).

Вариант 2 — локальный сервер:

```bash
python3 -m http.server 8000
```

и открыть `http://localhost:8000/`.

## Как поменять цифры тарифов

Все ставки, дашборды и тарифы — в одном файле `js/pricing-config.js`. Это обычный объект с данными,
логика расчёта его не хранит — можно менять суммы, названия дашбордов, состав категорий и квоты
без знания JavaScript, просто заменяя числа/строки в этом файле.

## Что реализовано

- Полный расчёт для 4 тарифов на дашбордах: Стартовый, Базовый, Продвинутый, Экспертный —
  выбор дашбордов из сетки 3 категорий, расчёт ежемесячного платежа и разового Install-платежа
  за дашборды/интеграции сверх квоты тарифа. Стартовый (подпись «для клиентов Фитбейс ПРО») —
  только для клиентов с подпиской Fitbase ПРО, 1 базовый дашборд включён бесплатно.
- Франшиза (Партнёр) — оплата за студию, единый Install-пакет (не зависит от числа студий) +
  интеграции с рекламными кабинетами по 25 000 ₽ каждая. Франшиза УК считается по тарифу
  «Кастомизированный».
- Кастомизированный — основной объект (5000 ₽/мес + Install по ТЗ в часах по 5000 ₽/ч +
  интеграция с рекламным кабинетом) и доп. объекты «Кастомизированный+» (только 2500 ₽/мес,
  без Install).
- Автогенерация текста коммерческого предложения, копирование в буфер обмена, экспорт в PDF
  через системный диалог печати браузера (кнопка «Скачать PDF») — доступно для всех 6 тарифов.

## Структура

- `index.html` — разметка страницы
- `css/styles.css` — стили (без внешних фреймворков и CDN, работает офлайн)
- `js/pricing-config.js` — данные тарифов и ставок
- `js/calculators.js` — чистые функции расчёта
- `js/state.js` — состояние выбора пользователя
- `js/render.js` — отрисовка DOM
- `js/offer-text.js` — генерация текста КП
- `js/pdf-export.js` — экспорт КП в PDF
- `js/main.js` — обработчики событий, точка входа
